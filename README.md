# Tubes2_GasAja
## SETUP
```bash
    cd src
```
for windows:
```bash
    ./setup
```
for mac:
```bash
    ./setup.sh
```


## RUN
```bash
    cd src
```
for windows :
```bash
    ./run
```
for mac :
```bash
    ./run.sh
```

open `localhost:3000`


## STOP
for windows :
```bash
    ./stop
```
for mac :
```bash
    ./stop.sh
```
